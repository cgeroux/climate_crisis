One meaningful way to take action is to talk about the climate crisis. A related way to take action is to put up posters. Putting up posters will make the issue more visible in the real world and it will make it harder for people to ignore the climate crisis. There are many ways to take action and finding ways to fit in bits of climate action in your everyday life will help make those actions a reality. The David Suzuki foundation has a list of ways you can take action and also information about how to have climate conversations, which isn't always easy.
If you would like to put up posters in your local area feel free to download, modify, and or print copies of this poster to put up in your area. The poster is licensed by CC0 1.0 Universal.
This poster was created using the free and open source vector graphics program Inkscape which can be used if you wish to modify the poster. A scalar vector graphic (svg) file of the poster is available here: climate_crisis.svg which is best if you want to modify it. If you just want to print the poster as it is, this pdf file is likely what you want.
